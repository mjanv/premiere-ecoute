# Wantlist Feature Plan

## Overview

Allow users to save tracks from active listening sessions into personal "wantlists" for later discovery. Users can create multiple named wantlists, add tracks from currently playing sessions, and manage their collections. Saved item can be a track, an album or an artist. Context `PremiereEcoute.Wantlists`.

## Use Cases

1. **Save from streamer**: Streamer is listening to a track and wants to remember it
2. **Browse wantlists**: User wants to see all their saved tracks across wantlists
3. **Manage wantlists**: Rename, delete, reorder wantlists
4. **Remove tracks**: Remove individual tracks from a wantlist

## Entry points

1. Currently Playing (Homepage Viewer View) display an "Add to Wantlist" button on the currently playing item (track/artist/album) on the stream
2. From any page of discography displaying tracks, albums or artist, add a "add to wantlist" option.
3. From the radio page when logged, each radio item display a "add to wantlist"
