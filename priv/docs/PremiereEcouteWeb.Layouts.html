    PremiereEcouteWeb.Layouts (Premiere Ecoute v0.1.0)

    View Source

    This module holds different layouts used by your application.

    See the layouts directory for all templates available. The "root" layout is a skeleton rendered as part of the application router. The "app" layout is rendered as component in regular views and live views.
